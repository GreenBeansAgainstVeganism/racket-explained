Bracket Helper

One of the biggest issues people encounter with learning to code in Racket is code readability. Because of Racket's unconventional syntax, and because the style guide is a little convoluted, many students tend to forego all formatting practices entirely. This tends to backfire epically later in the form of requiring you to go on several wild goose chases trying to track down a mismatched pair of parentheses for half an hour, so I created this tool to help you recover from these situations.

Paste your Racket code into the box below and hit "Go!" to highlight it so you can easily see where things aren't lining up:

Tip: Although you can't auto-format, you can press Ctrl + I to auto-indent your file in DrRacket.
